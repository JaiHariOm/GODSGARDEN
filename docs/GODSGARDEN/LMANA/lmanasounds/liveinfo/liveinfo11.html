LES PETITES FETES Vol.14

Event info

LES PETITES FETES Vol.14開催のお知らせです！

☆本イベントは、今秋フランス・パリで開催する100% Love%Peace Paradeのプレイベントとして運営しています☆

今回のプティフェテは、映像作家の岸本智也さんの創り出す空間を皆さまにお届けしたいとおもいます。

複数のプロジェクターを使用してFlow+輪をどんな色に染めていただけるのか、期待が高まります。

その空間に、3色の音楽を組み合わせて、どんな反応を起こすのか、皆さまの五感でお楽しみください。

皆さまのご参加をお待ちしております。

イベント概要

【会場】

Flow+輪（フラワー）

東京都渋谷区松濤１丁目１４７

【予約先】

(下記アドレスにお名前・連絡先・人数をメールしてくださいね) info.ruuruu@gmail.com

【開催日時】

2011年6月11日（土）
open:18時～ start:19時～

☆ライブ終了後に交流の時間があります。(各自お好みでお料理や飲み物をお求めいただき、お楽しみください♪)

【チャージ】

2000円+500円（ウェルカムドリンク代+チャリティ）
中学生以下は無料！

出演者プロフィール

★Tomoya Kishimoto（映像）

2005年より活動開始。
類まれなセンスと想像力によって生み出される空間演出は数々の企画で高い評価を得る。
ときには10台以上にも及ぶプロジェクターを巧みに操り、
音の中に映像がメルトする瞬間は、独自の世界観のみ成し得る賜物かもしれない。

OM SHANTI OM


☆輝鷹

2006年バンド活動を経て、ソロに転身。
現在はアコースティックギター一本での演奏、弾き語り、 ジャンルレスにダンサーやミュージシャンとセッションを行っている。
クラブ、ライブハウス、カフェ、寺社、野外イベントにてライブを行う。
誰の心の中にでもある、柔らかい部分にふれるような詩と旋律を謳い、様々な風景を映し出すギターを奏でる。
また、カリンバ、打楽器奏者Sageとのユニットhawkcrower(ホウクロア)としても活動中。

★林圭樹（ネイティブアメリカンフルート）

HAYASHI KEIJU

古代からの神聖幾何学の教えに触れ、開く。
光の乗り物MERKABAを使う心の冒険者。
気功士として人々の心に元気を取り戻し、目覚めをサポートを行う傍ら、 コズミックトライバルバンド『LAGO』のベーシストとして心・体を踊らせる活動も。
今回は心を込めてネイティブアメリカンフルートで、日本からの祈りを発振します。

☆a-sha+（A.K.A山根麻以）

80年代に９枚のソロCDをリリース。
90年にフリーランスとなり、世界を旅する傍ら様々なコラボやセッションを展開。
97年、活動の拠点を富士山麓に移し"New Archaic Sounds" を設立。
『ふつうの唄』で新たな境地を拓き 人間の本質を謳った作品を作り続けている。
自身の活動のすべてを「 日本列島やさしいきもち化計画」と位置づけ コンサートツアー、教育の現場など活躍の場は幅広い。
愛・地球博、パリ、サラゴザなど海外公演も多く、 人種や宗教を超えた感動的なライブを各地で展開している。
魂が響き合うような、クリアで愛にあふれる一体感は他に類がない。
